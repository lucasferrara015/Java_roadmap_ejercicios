## 📖 Tipos de datos en Java
- **Primitivos**:
  - `int` → enteros
  - `double` → decimales
  - `char` → caracteres
  - `boolean` → valores lógicos (`true` / `false`)
  - Otros: `byte`, `short`, `long`, `float`

- **No primitivos**:
  - `String` → texto
  - Clases y objetos definidos por el programador

---

## 📖 Variables
- Una **variable** es un espacio en memoria donde se guarda un valor.  
- Se declara indicando tipo + nombre + valor:
  ```java
  int edad = 25;
  String nombre = "Lucas";
  ```
  ---

##   Operadores aritméticos
Los operadores aritméticos permiten realizar cálculos básicos con variables numéricas:
- + → suma

- - → resta

- * → multiplicación

- / → división entera

- % → módulo (resto de la división)
