# Java Roadmap Ejercicios 🚀

Incluye ejercicios vistos en el curso **TodoCode Academy** + prácticas personales.  
Además, agrego apuntes teóricos y un roadmap de aprendizaje progresivo.

---

## 📖 Roadmap Java
1. **Tipos de datos y variables**
   - Declaración, inicialización, operadores básicos.
2. **Estructuras selectivas (condicionales)**
   - if, else, switch.
3. **Estructuras repetitivas**
   - for, while, do-while.
4. **Arrays: vectores y matrices**
   - carga, recorrido, búsqueda, ordenamiento.
5. **Modelo de capas + IGU (Swing)**
   - separación lógica, formularios, botones.
